# MPP-NEAR Website

Informational website for the Machine Payments Protocol implementation on NEAR blockchain.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Deployment

This website is deployed on Vercel: https://mpp-near-website.vercel.app

## Structure

- `app/` - Next.js App Router pages
- `components/` - React components
- `public/` - Static assets

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
